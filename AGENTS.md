# Notes

- WorkOS TanStack Start currently installs from `@workos/authkit-tanstack-react-start`. Some newer docs and examples refer to `@workos-inc/authkit-tanstack-start`, but that package was not published to npm when this integration was refreshed.
- `@tanstack/ai@0.9.1` supports the repository agent loop cleanly via `chat()` + `toolDefinition()` + `maxIterations()`, but the top-level `chat()` activity options do not accept a `request` field even though lower-level `TextOptions` types include one.
- The repository AI runtime is currently duplicated in `packages/agent-runtime` and `apps/repo-actions/src/runtime.ts`; they have already drifted. The worker copy parses tool call input, while the shared package currently records `TOOL_CALL_START` inputs as `{}`.
- Creating a task in the web app immediately requests AI planning. The current product flow therefore couples task creation to AI enrichment instead of treating AI as an optional layer on top of a stable task model.
- The planner-first rebuild assumes AI suggestions are never canonical by default, sub-issues are normal child tasks, discussion-triggered rereview is explicit, and workspace AI quota enforcement is a hard stop for new runs.
- `effect` currently publishes stable `3.21.0` and a V4 beta line (`4.0.0-beta.*`). If the rebuild keeps the explicit "Effect V4" requirement, install a beta release instead of assuming a stable 4.x package exists.
- The web app's WorkOS callback route is implemented at `/auth/callback`. If auth starts behaving inconsistently after sign-in, check that `WORKOS_REDIRECT_URI` matches that exact path and current local dev port; the older `/api/auth/callback` shape and stale port values break the session bootstrap in non-obvious ways.
- The shared `useJson()` helper used across onboarding and workspace pages used to collapse backend exceptions into a bare `fetch failed`. When debugging workspace bootstrap, inspect the rendered page error or platform route response body first; server-side Convex and auth errors often surface there.
